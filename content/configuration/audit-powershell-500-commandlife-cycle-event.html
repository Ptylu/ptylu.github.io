Configuration

This Event logs Powershell commands launched with powershell interface or script. There are at leat 2 methods for the powershell eventlog 500 activation

1st method admin-right

Create file C:\\Windows\\System32\\WindowsPowerShell\\v1.0\\profile.ps1 and add $LogCommandLifeCycleEvent=$true. Needs a reboot to be effective

This configuration stays after reboot and is effective on all users.

2nd method user-right

Open powershell and launch this command $LogCommandLifeCycleEvent=$true. Doesn't need a reboot to be effective

This configuration disappears after reboot and is effective only on user that launched command.


Attention The second method can be used by non-admin user to avoid Powershell logging during its current session.
